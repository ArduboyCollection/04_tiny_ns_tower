Tiny NS-Tower for Arduboy

## installation
1. pls edit test.bat for your computer(must change COM port number)
2. execute test.bat
3. play it


## Rule
player climb tower. goal is 30 floor.


## Controls
A / left : jump
up       : reset


## My development
compiler        : windows Arduino IDE 1.6.7(avr-gcc) + 1.0.6(make.exe)
image converter : python 2.6 + PIL
etc             : Visual Studio 2015 C#
